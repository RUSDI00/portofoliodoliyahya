Tentang Saya

Doli Yahya Ritonga

Nama saya Doli Yahya Ritonga, seorang mahasiswa aktif di Program Studi S1 Teknologi Informasi, Universitas Labuhanbatu, yang sedang berproses menjadi profesional IT yang tidak hanya kompeten secara teknis, tetapi juga matang secara karakter, kepemimpinan, dan visi.

Saya lahir pada 2 Agustus 2004 di Tanjung Selamat, sebuah wilayah yang sarat dengan nilai-nilai lokal, kebersamaan, dan kerja keras yang telah menjadi bagian integral dari identitas saya.

Saya adalah anak keempat dari lima bersaudara. Tumbuh besar dalam keluarga sederhana mengajarkan saya arti penting dari tanggung jawab, komunikasi efektif, dan kebersamaan. Dari lingkungan keluarga inilah saya belajar untuk mendengarkan, memberi ruang untuk ide orang lain, dan berlatih menjadi pemimpin yang mengedepankan kolaborasi daripada kompetisi.

Perjalanan Akademik dan Minat Profesional

Minat saya terhadap dunia teknologi dimulai sejak masa sekolah menengah pertama, ketika saya mulai mengeksplorasi komputer, jaringan, dan pemrograman dasar secara otodidak. Rasa ingin tahu itu berkembang menjadi komitmen akademik, yang akhirnya membawa saya ke dunia pendidikan tinggi di bidang Teknologi Informasi.

Saya percaya bahwa teknologi bukan sekadar alat bantu, melainkan medium strategis untuk menghadirkan solusi nyata bagi persoalan masyarakat, termasuk dalam bidang pendidikan, kesehatan, pemerintahan, dan lingkungan.

Sebagai mahasiswa TI, saya tertarik secara mendalam pada beberapa topik kunci:

  • Pengembangan Perangkat Lunak (Software Engineering)
  • Analisis dan Desain Sistem Informasi
  • Jaringan Komputer dan Keamanan Siber (Cybersecurity)
  • Kecerdasan Buatan (AI) dan Pembelajaran Mesin (Machine Learning)
  • Manajemen Proyek Teknologi (IT Project Management)

Saya aktif dalam berbagai kegiatan akademik dan non-akademik yang melatih kemampuan problem-solving, critical thinking, serta kolaborasi tim lintas disiplin. Beberapa proyek yang telah saya kerjakan melibatkan analisis kebutuhan sistem, pembuatan aplikasi berbasis web, serta penerapan prinsip-prinsip UI/UX untuk meningkatkan pengalaman pengguna.

Filosofi Belajar dan Komitmen Profesional

Saya memegang teguh filosofi bahwa belajar adalah proses seumur hidup. Di tengah perubahan teknologi yang sangat cepat, saya berkomitmen untuk selalu belajar, berkembang, dan beradaptasi. Saya rajin mengikuti kursus daring, seminar, webinar, dan komunitas pengembang teknologi untuk memperluas wawasan dan jaringan profesional saya.

Saya percaya bahwa pembelajaran yang berkelanjutan adalah kunci untuk tetap relevan dan kompetitif di dunia kerja masa kini dan masa depan.

Saya terbiasa menggunakan berbagai alat bantu profesional seperti:

Figma untuk desain UI

Visual Studio Code untuk coding

Trello dan Notion untuk manajemen proyek

Git dan GitHub untuk kolaborasi kode

Karakter dan Soft Skill

Teknologi tidak bisa berdiri sendiri tanpa dukungan keterampilan interpersonal dan etika profesional. Karena itu, saya juga melatih diri dalam hal-hal berikut:

Kepemimpinan

Mampu mengambil inisiatif dan mengoordinasi tim dengan bijak

Komunikasi Efektif

Menyampaikan ide secara jelas, baik lisan maupun tulisan

Adaptabilitas

Cepat belajar hal baru dan menyesuaikan diri dalam berbagai situasi

Ketekunan dan Komitmen

Tidak mudah menyerah dalam menghadapi tantangan teknis maupun non-teknis

Saya meyakini bahwa kombinasi antara kompetensi teknis dan etika kerja profesional adalah kunci untuk menciptakan solusi teknologi yang bukan hanya canggih, tetapi juga bermanfaat dan berkelanjutan.

Visi Jangka Panjang

Visi saya adalah menjadi seorang teknolog strategis yang mampu menjembatani kebutuhan masyarakat dengan solusi teknologi yang relevan dan berdampak nyata. Saya ingin menjadi bagian dari ekosistem inovasi digital di Indonesia, terlibat dalam transformasi digital di sektor publik maupun swasta, serta berkontribusi dalam membangun ekonomi digital yang inklusif dan adil.

Dalam jangka panjang, saya bercita-cita untuk:

  • Membentuk startup teknologi berbasis solusi lokal
  • Menjadi konsultan IT atau analis sistem di perusahaan teknologi ternama
  • Memberdayakan masyarakat melalui program pelatihan digital literacy
  • Terlibat dalam riset dan pengembangan teknologi berbasis kecerdasan buatan
